About Project

On the front page there is a submission form and in the back there is an admin panel where one can edit and delete customer reservations as well as menu items. I used an html template for the front end to save time and just made small ajustments so it can work with my backend, same goes for dashboard ui.

@pc_setup_ideas Instagram Bot

View Source Code
Python SQLite Optical Word Recognition Facial Recognition Image Fingerprinting

About Project

The project is an Instagram bot, @pc_setup_ideas, dedicated to sharing PC setup ideas. In just one year, it has amassed an impressive following of over 32,000 users while operating on a Raspberry Pi, demonstrating its efficiency and scalability. The follower count has since fallen due to inactivity.

The bot's workflow is structured for content quality and user engagement. It begins by monitoring the image count and, if it falls below a set threshold, it gathers new images from selected instagram pages, moves those new images to a quarantine directory. Here, it rigorously checks for duplicates, human faces, and red flag keywords to filter out advertisements. Images that fail these checks are promptly deleted, while the rest are hashed for future duplicate checks and stored in the "future_images" directory.

Once this directory is prepared, the bot randomly selects an image and retrieves a random description and 25 relevant hashtags from its database. It then posts this content to Instagram using Selenium script (at the time of making, Instagram didn't have an API to post content) and logs bot data before waiting for a specified duration to repeat the cycle. This systematic approach ensures consistent, high-quality content delivery, contributing to its impressive follower growth.

functionality

  • Taking advantage of Selenium to post images

  • OCR used for filtering advertisements

  • Hosted on my Raspberry Pi 3B+

  • Facial recognition used to filter out images with human faces

  • Image hashing (or image fingerprinting) used to filter out already posted images
